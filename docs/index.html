~/m3l-automation — about

A shared infrastructure library for automation scripts and AWS Lambda handlers, written in TypeScript 6 / ESM-only, Node 24+. It provides application scaffolding, configuration, structured logging, error handling, file I/O, polling and retry resilience, and AWS credential management — so consumer scripts stay free of boilerplate.
